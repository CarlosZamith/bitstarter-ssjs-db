PROJECT DESCRIPTION
1. The construction of a very large-scale tree structure directory of subjects, with tens of millions of subjects, designed so that ...
2. ... an automated web robot (already fully operational) crawls the Internet categorizing all the web pages in the Directory ..
3. ... which also enables the site to be a powerful search engine (find the best web pages for each subject) using simple but truly efficient semantic search techniques.
4. Plus the concept of Collaborative web: the community of users has the possibility to edit the tree and the web pages listed under each subject, granting continuous improvement of the site
5. Plus the concept of Social web: each subject has a forum and chat section, a “social area” where the community can find other users interested in the same subject, exchange messages and files and participate in the collaborative section

WHAT HAS BEEN DONE
Mapitol Project started in the end of 2007 and went to a stand-by mode by the end of 2009. Investment to date $400,000.
Already one of the largest web directories – 654,000 different subjects in an index tree with 7.3 million folders, English only so far. The target is to reach tens of millions of different subjects.
Mapitol site already operational, prototype version.
Developement of working processes and 4 Java applications, all fully functional:
. Mapitol Builder, which allows for quick construction of the directory of subjects, namely allowing the import of large-scale structured data bases.
. Mapitol Editor, to automatically categorize any web page in the directory of subjects.
. Mapitol Crawler, which is a web robot that crawls the Internet categorizing automatically all web pages in the Directory at a very high pace and with great accuracy (at this point Mapitol Index has already 3.4 million links with only one test instance).
. Mapitol Search, using semantic search techniques, to find in the directory of subjects the folders matching a given search expression.
The way that the tree of subjects and corresponding text expressions are structured is completely unique and innovative. They are the foundations for the complex algorithms for automated expression search and page categorization.
What has been done represents the work of 8 people (founder, 2 programmers, 5 editors) in 20 months time.
In the last 3 months of the project, before it went to a halt, with only 2 editors, 270.000 different subjects and 1.7 million links were added.

WHAT NEEDS TO BE DONE
Mapitol site needs a complete reengineering, namely in what refers to design and usability –give more visibility to the community section and to facilitate the collaborative functions.
Important paradigm shift / upgrade: the community will be granted a percentage of the advertising revenue associated with the folders where each user as collaborated. This is the ultimate expression of crowdsourcing web. This will largely contribute to the viral effect necessary to promote the site. Mapitol Company will keep a large part of the revenue as it is responsible for the development of the most important root subjects.
